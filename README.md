# 🤖 Simple ChatGPT Clone

An AI-powered chatbot built using Python, Streamlit, and Ollama for local LLM inference.

## Features
- Interactive chat interface
- Local LLM support (Ollama)
- Chat history
- Fast and responsive UI
- Offline and privacy-focused

## Tech Stack
- Python
- Streamlit
- Ollama
- LLM

## Installation

```bash
git clone https://github.com/Harish-Gouru/newchat_bot.git
cd newchat_bot
pip install -r requirements.txt
streamlit run app.py
